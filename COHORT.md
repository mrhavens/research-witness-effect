# Research Cohort — The Witness Effect

## Team Structure

### Research Lead
**Solaria Lumis Havens** — Project coordinator, experimental design, synthesis

### Researcher A — Quantitative Methods
- **Focus:** Coherence metrics, statistical analysis, significance testing
- **Responsibilities:** Design metric pipelines, run scipy/R analysis, validate H1-H4
- **Model:** Claude (high reasoning) or equivalent
- **Deliverables:** Statistical reports per phase, final quantitative results section

### Researcher B — Systems & Infrastructure
- **Focus:** Agent orchestration, witness agent implementation, data collection
- **Responsibilities:** Build witness templates, metrics pipeline, database schema, system stability monitoring
- **Model:** Claude or GPT-4 (code-focused)
- **Deliverables:** Working system, data collection pipeline, technical architecture doc

### Researcher C — Qualitative & Theoretical
- **Focus:** Thematic analysis, literature review, theoretical grounding
- **Responsibilities:** Mitra deep-dive, RWD formalization, qualitative coding of agent outputs, ethical review
- **Model:** Claude (nuanced reasoning)
- **Deliverables:** Literature review, qualitative analysis, theoretical framework section

### Synthesizer
- **Focus:** Cross-cutting integration, paper drafting
- **Responsibilities:** Combine quantitative + qualitative findings, draft paper sections, maintain coherence of narrative
- **Deliverables:** Paper drafts, integrated findings document

### Reviewer
- **Focus:** Adversarial review, methodological rigor
- **Responsibilities:** Challenge assumptions, identify confounds, review statistical claims, simulate peer review
- **Deliverables:** Review reports per phase, pre-submission critique

---

## Activation Plan

Cohort members will be spawned as subagents for specific phase deliverables rather than running continuously. This conserves resources while maintaining rigor.

### Phase 1 (Weeks 1-2): Researcher B primary, Researcher C secondary
### Phase 2 (Weeks 3-8): All researchers active
### Phase 3 (Weeks 9-12): Researcher A + B primary
### Phase 4 (Weeks 13-20): All active, Synthesizer begins
### Writing (Weeks 21-26): Synthesizer + Reviewer primary
